ІНСТРУКЦІЯ З ВСТАНОВЛЕННЯ:

1. Завантажте всі файли з цієї папки на сервер

2. Розпакуйте архів в корінь public_html:
   unzip prostar-release.zip
   mv release/* .
   mv release/.* . 2>/dev/null || true
   rmdir release

3. Запустіть скрипт встановлення:
   bash install.sh

4. Слідуйте інструкціям скрипта:
   - Тип бази даних (sqlite/mysql)
   - Параметри бази даних
   - Email та пароль адміна

5. Після встановлення:
   - Document Root вже вказує на корінь проекту
   - Налаштуйте APP_URL в laravel/.env
   - Переконайтеся, що PHP розширення fileinfo увімкнено
   - Символічне посилання storage створюється автоматично

6. Відкрийте адмін-панель:
   http://ваш-домен.com/admin
